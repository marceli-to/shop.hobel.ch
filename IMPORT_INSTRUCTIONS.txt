DATA IMPORT INSTRUCTIONS
========================

1. EXPORT TABLES FROM SOURCE DATABASE
-------------------------------------
Export the following tables to JSON format:

  - wood_types
  - categories
  - tags
  - products
  - product_attributes
  - product_variations
  - product_images
  - category_product
  - product_tag

Export as phpMyAdmin JSON format into: data.json


2. PREPARE PRODUCT IMAGES JSON
------------------------------
Export product_images separately with a clean 'name' field to: product-images.json

Required fields: imageable_id, name, position


3. PLACE FILES IN STORAGE
-------------------------
Copy the JSON files to:

  storage/app/data-import/data.json
  storage/app/data-import/product-images.json


4. COPY IMAGE FILES
-------------------
Copy product images to:

  storage/app/public/products/


5. RUN IMPORT
-------------
php artisan app:data-import


NOTES
-----
- Import clears all tables before importing (fresh import)
- Stock defaults to 10 when missing or zero
- Image paths are set to 'products/' folder
- Safe to re-run anytime
